Welcome to Quiz App

I have created simple quiz app 

Features:
1. Each question have 10 seconds to answer and total time of the section will be 10* Number of question after the  time ends the test will automatically submitted.
2. To end the test manually we have to answer all the question.
3. If the time ends the test will automatically ends, weither the user not answered all the question.
4. We can traverse any question at any time and we can change the submitted answers also.
5. After submission User will get the score and if they want they can take the quiz once again.
